# MERN Social Web Application

A full-stack social networking web application built using **MongoDB, Express.js, React.js, and Node.js**. This project enables users to create accounts, connect with others, manage profiles, and interact through a request-based matching system. It focuses on secure authentication, scalable backend architecture, and a smooth user experience.

## 🚀 Features

* Secure **Signup / Login Authentication** using JWT & Cookies
* Password encryption with **bcrypt**
* **Forgot Password** flow with OTP Email Verification
* OTP expiry validation for enhanced security
* Strong password validation before reset
* View and Edit Profile functionality
* Send Connection Requests (Interested / Ignore)
* Accept / Reject incoming requests
* Prevent duplicate connection requests
* View received requests and accepted connections
* Personalized user feed with filtering logic
* Pagination support for scalable feed loading
* Fully responsive frontend built with React.js
